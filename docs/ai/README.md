## Core AI Decision

V1 should use AI as a lightweight assistant layer.

AI can support:

- writing feedback
- speaking feedback
- recommendation explanations
- learner insights
- summaries

AI should not own:

- roadmap structure
- core progress logic
- review triggers
- product state
- database schema
